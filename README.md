# cc4.marketing Website

This repository contains the source code for the [cc4.marketing](https://cc4.marketing) website.

## Tech Stack

- **Framework:** Astro
- **Deployment:** GitHub Pages

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start dev server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
