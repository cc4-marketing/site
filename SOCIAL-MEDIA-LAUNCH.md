### r/ClaudeAI

**Title:** I built a free interactive course to learn Claude Code for marketing work

Hey everyone,

I've been using Claude Code for marketing work and realized there's no good resource for marketers to learn it properly. So I built one.

**What makes it different:**

Instead of watching videos or reading docs, you learn inside Claude Code itself. You download the course materials, open them in Claude Code, type `/start-0-0`, and Claude guides you through each lesson.

**What's covered:**

- Module 0: Getting started with Claude Code
- Module 1: Fundamentals (working with files, agents, project memory)
- Module 2: Marketing workflows (briefs, content strategy, copy, SEO, competitor analysis)

**It includes:**

- 16 interactive lessons
- A practice project (you're a marketing agency working for a B2B SaaS client)
- Marketing plugins for campaign workflows

**Why free?**

I genuinely believe AI tools should help marketers work smarter. The best way to learn is hands-on, and I wanted to lower the barrier.

Check it out: cc4.marketing

Happy to answer any questions or take feedback. This is v0.1.0 so there's definitely room to improve.
